1
O Começo

O que é Programação?

▼

Programar é apenas dar instruções para o computador (ou celular) resolver problemas para nós. É como escrever uma receita de bolo: você diz os ingredientes e o passo a passo que o aparelho deve seguir.

Recomendação básica: 📺 O que é um Algoritmo? (Vídeo - Curso em Vídeo) ↗
2
Lógica

Lógica de Programação

▼

Antes de escrever códigos complicados, precisamos aprender a organizar nossos pensamentos de forma que um computador entenda. É aqui que você aprende comandos como "se" (if) e "enquanto" (while).

Para começar forte: 📺 Curso Grátis de Algoritmos (Vídeo - Guanabara) ↗
3
Web Básico

HTML e CSS: A Cara da Web

▼

O HTML cria a estrutura do site (os textos, caixas e botões), e o CSS deixa tudo bonito (cores, fontes e espacamento). É como construir uma casa: HTML são os tijolos e CSS é a pintura.

Onde aprender de verdade: 📺 HTML5 e CSS3 para Iniciantes (Vídeo - Guanabara) ↗
4
Dinamismo

JavaScript: Dando Vida ao Site

▼

O JavaScript faz as coisas acontecerem de verdade: calcular resultados, abrir menus de celular, enviar formulários e criar joguinhos. É o cérebro das páginas da web.

Melhor curso em português: 📺 Curso de JavaScript (Vídeo - Guanabara) ↗
5
Especial

Onde ir a partir daqui?

▼

Parabéns por chegar até aqui! Agora você já sabe o básico. O segredo é praticar criando pequenos projetos (como o seu próprio site de apresentação) e participar de comunidades focadas em iniciantes para tirar dúvidas.

Guia Prático

Estudar sem computador?
Sim, é totalmente possível!

Muitos programadores começaram apenas lendo apostilas e praticando em celulares antigos. Não deixe a falta de equipamento te parar. Veja algumas formas de dar a volta por cima:

A

Pratique no Papel

Escrever algoritmos e códigos em um caderno ajuda seu cérebro a memorizar a sintaxe sem as correções automáticas das ferramentas.

B

Use Apps Leves

Existem aplicativos incríveis na Play Store e App Store como o Treedit (para criar páginas HTML) e o Sololearn (que tem pequenas lições de código).

C

Baixe os Vídeos no Wi-Fi

Se a sua internet móvel for limitada, use o Wi-Fi de praças públicas, bibliotecas ou de conhecidos para baixar os vídeos do YouTube offline e estudar depois.
